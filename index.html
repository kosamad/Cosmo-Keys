How To Play

The aim of the game is to correctly type the same letter or word as the computer.

The timer will go for 1 minute, how many can you get?

About Cosmo Keys

This game is a typing challenge! It’s been created with visually impaired users in mind, but anyone can use it to practise their typing skills.

On the next page you can select your level, each corresponds to the number of letters challenged e.g level 1 will give 1 letter at a time. You'll then be taken to the game page.

After you press the play button, the game will read each challenge to you. If you have selected Level 1 or Level 2 you will be read each leter in turn. On level 3 you are given a 3 letter word to type.

At the end of the game your score is announced. Can you beat it next time?
