    The aim of the game is to correctly type the same letter (level 1) or word (level 2+) as the computer.

    The timer will go for 1 minute, how many can you get?

  • About
    • How To Play

    This game is a typing challenge! It’s been created with visually impaired users in mind, but anyone can use it to practise their typing skills.

    The game reads each challenge to you and will announce your score at the end. On the next page you can select your level, each corresponds to the number of letters challenged at a time e.g level 1 will give 1 letter before waiting for your typed response.
